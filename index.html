Mis Proyectos

Landing page

Mi primer proyecto para Rock {The Code} fue un clon de la web de Imdb, usando únicamente HTML y CSS.

El mayor reto en este proyecto fue la creación de elementos dinámicos como un carousel y un menú hamburguesa sin usar js.

Tienda con filtro

En el segundo proyecto, incorporamos js a la mezcla. El objetivo era crear una tienda de manera dinámica a partir de una pseudo-base de datos (un objeto) y luego agregar filtros de búsqueda.
En este caso, la víctima fue PC Componentes.

Selector de colores

Con este pequeño ejercicio puse en práctica el uso de Vite y js DOM.

Con esta aplicación se puede elegir un color predeterminado de una lista o bien de una paleta y ver cómo queda en el fondo y cuál es su código HEX.

Password checker

Esta mini app desenfoca la imagen de fondo dependiendo del largo de la contraseña.
